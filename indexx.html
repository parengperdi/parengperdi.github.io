Device Overview

SPY (Smart Protective Yondu) is a device that employs a smart whistle that signals and alerts pre-selected emergency numbers through SMS Notification. The device is a communication device that alerts people by sending the owner's location using a GPS tracker.

Arduino Nano Board

The Arduino Nano was first released in 2008 and is one of the most popular Arduino boards available. The Nano is a breadboard-friendly board, based on the ATmega328 8-bit microcontroller by Atmel (Microchip Technology). It has the same functionality as the Arduino Uno but in a smaller form factor.

LEARN MORE

A9G GSM/GPRS+GPS Module

A9G is a complete quad-band GSM/GPRS+GPS module based on the RDA8955 chip. The cost reduction of the core chip provides users with a cost-effective IoT solution. Integrating protocol stacks such as GSM/GPRS inside, this module supports the basic phone voice call/SMS, serial to GPRS, and GPS data transmission functions, which can be used in a wide range of applications, such as IoT, vehicle-mounted equipment, security devices, remote localization, electric power environment monitoring.

LEARN MORE

Sound Sensor Module

A sound sensor is defined as a module that detects sound waves through their intensity and converts it to electrical signals. Sound detection sensor works similarly to our Ears, having a diaphragm that converts vibration into signals. However, what’s different is that a sound sensor consists of an in-built capacitive microphone, a peak detector, and an amplifier (LM386, LM393, etc.) that’s highly sensitive to sound.

LEARN MORE

Battery and Battery holder

Lithium coin CR2032 batteries are the most common battery coin providing long-lasting, reliable power for various devices. They are used to power small electronic devices such as calculators, wristwatches, various medical devices, fitness appliances, toys, etc. The researcher will be using two cr2032 to power the Arduino nano since it requires a power of 5V to operate.
